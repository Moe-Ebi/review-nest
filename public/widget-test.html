ReviewNest — Widget Test Page

Demo mode: the widgets below use data-demo (sample reviews, no API call). To test with real data, replace with data-location-id="YOUR_LOCATION_ID" and remove data-demo.

1. Luxury Trust Carousel — light (default)

Desktop: 3 cards · Tablet: 2 cards · Mobile: 1 card + swipe. Arrows, dots, keyboard navigation, "Read more" on long reviews.

2. Carousel — dark mode

Same widget with data-theme="dark" on a dark host section.

3. Compact trust badge

For headers, product pages, cart and checkout. Click to open the review panel.

4. Review wall (masonry grid)

For dedicated review pages: data-mode="grid".

5. Custom branding via CSS variables

Warm brand accent, amber stars kept, tighter radius, stronger shadows — all via data-* attributes.

6. Legacy embed format (existing client installs)

The old <script data-location> format keeps working — this one also uses data-demo for testing.

Embed snippet (copy for clients)

<div id="reviewnest-widget"
     data-location-id="YOUR_LOCATION_ID"
     data-mode="carousel"></div>
<script src="https://reviewnest.ebiconsulting.co.za/widget.js" async></script>
